1. Провести полное тестирование сайта https://www.sharelane.com по чеклисту из предыдущего ДЗ  
2. На регрессионное тестирование в qase.io:
- добавить тест-кейсы, можно только с названием
- добавить Test Run, проставить статусы
- все найденные баги завести и связать с Test Run (title + severity)
4. После окончания тестирования стори перевести в статус ‘Complete’
5. Cгенерировать репорт по результатам тестирования и пошарить репорт нажам кнопку "Share report"
Ссылку на репорт можно скинуть в телеграмм или добавить файл на гит, в котором будет ссылка
